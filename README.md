# ⚾ Chicago Cubs S&C Dashboard

Athlete Readiness & Performance Dashboard for Chicago Cubs Strength & Conditioning.

## Access

**[Open Dashboard →](https://markweisman45.github.io/cubs-sc/)**

## Features

- Athlete readiness tracking (CMJ, Statcast, wellness)
- Program builder (single session, weekly, mesocycle)
- Periodization planning tool
- Exercise library with video links
- Team overview & ACWR monitoring
- Coach notes per athlete
- Data backup & cloud sync (Supabase)

## Data Storage

All data saves to browser localStorage automatically.
Use the **Download Backup** button regularly to keep a safety copy.
